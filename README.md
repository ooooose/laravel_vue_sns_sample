# laravel_vue_sns_sample
#### Laravel✖️Vueの練習アプリ
TechPitの教材で練習したアプリです。
## 機能概要
- ユーザー登録・ログイン機能（Google認証）
- 記事投稿・編集・削除機能
- いいね機能
- タグ機能
- フォロー機能
